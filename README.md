# 🚀 Support Triage Intelligence System (STIS)

- 🧑‍💻 HackerRank  
- 🤖 Claude  
- 💳 Visa

- Risk-aware decision engine  
- Safe escalation handling  
- Document-grounded responses (no hallucination)
